AIHabitBuilder
AI-powered habit builder
- - Deployment link: https://staging.dxkb0ao356ox0.amplifyapp.com/
- User inputs a habit they want to develop and the app uses AI to generate small, actionable steps the user can take to achieve that goal!
- This full-stack AWS AI Habit Planner web application was built using AWS Lambda, API Gateway, DynamoDB, and Amazon Bedrock for backend while the frontend was created with AWS Amplify
